# Patrones Flamenca

Generador interactivo de patrones de costura con especialidad en moda flamenca.

## Funcionalidades

- **6 plantillas parametrizadas**: Corpiño, Falda de Volantes, Volante Circular, Manga, Falda Godets, Bata de Cola
- **Medidas personalizables**: Introduce tus medidas y el patrón se genera automáticamente
- **Marcas de costura**: Pinzas, piquetes, hilo de la tela y líneas de referencia
- **Guía de corte**: Instrucciones detalladas de cómo colocar la tela para cortar cada pieza
- **Exportar SVG a escala 1:1**: Para imprimir en plotter o impresora de gran formato
- **Imprimir directamente**: Desde el navegador

## Uso

Abre `index.html` en cualquier navegador moderno. No necesita servidor, instalación ni dependencias.

## Tecnología

Aplicación React renderizada en el navegador con Babel standalone. Completamente autónoma en un solo archivo HTML.
